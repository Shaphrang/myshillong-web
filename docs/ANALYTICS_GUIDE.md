# Analytics Guide
`marketplace_events` is the event ledger for:
- vendor_view
- deal_view
- phone_click
- whatsapp_click
- website_click
- direction_click
- ad_impression
- ad_click
- deal_save

Counters are denormalized on vendors/deals/placements for fast dashboard reads.

Use analytics views to report ROI to vendors and justify premium/featured plans.
